What-If Analysis (6 Points)
	1.	What-If Analysis is a feature in Microsoft Excel used to check how changes in input values affect the result of a formula.
	2.	It helps users analyze different possibilities or outcomes.
	3.	It is commonly used in business forecasting, budgeting, and profit analysis.
	4.	It allows users to modify variables and observe the result automatically.
	5.	What-If Analysis includes tools such as Scenario Manager, Goal Seek, and Data Table.
	6.	It helps in decision making by comparing different situations.

⸻

Steps to Perform What-If Analysis in Excel
	1.	Open Microsoft Excel and enter your data and formula.
	2.	Click on the Data tab in the ribbon.
	3.	In the Data Tools group, click What-If Analysis.
	4.	Select a tool such as Scenario Manager, Goal Seek, or Data Table.
	5.	Enter the required values or scenarios.
	6.	Excel will calculate and display different possible results.

⸻

2. Scenario Manager (6 Points)
	1.	Scenario Manager is a tool available inside What-If Analysis in Excel.
	2.	It is used to create and store multiple sets of input values called scenarios.
	3.	Each scenario represents a different situation such as best case, worst case, or expected case.
	4.	It allows users to compare results from different scenarios easily.
	5.	It is useful in business planning, financial analysis, and forecasting.
	6.	Scenario Manager helps in analyzing how changes in variables affect the final result.

⸻

Steps to Use Scenario Manager in Excel
	1.	Open Microsoft Excel and enter the data with a formula.
	2.	Click the Data tab.
	3.	Click What-If Analysis.
	4.	Select Scenario Manager.
	5.	Click Add and enter a scenario name.
	6.	Select the changing cells, enter values, and click OK.
	7.	Repeat to create multiple scenarios.
	8.	Click Show to view results for each scenario.
